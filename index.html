IBM Verse Developer Browser Extension

IBM Verse Developer Extension is a browser add-on which is used in the development of widgets for IBM Verse. It allows you to register your widget in IBM Verse as you develop it. Widgets registered with the extension are only available in the developers' local Chrome browser, and cannot be accessed from other devices.

View the Project on GitHub IBM-Verse/verse-developer-chrome-ext
